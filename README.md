## 5. 員工事項
- 招聘布條
- 加保：勞保、健保、勞退  
- 薪資結構設定（時薪／月薪）  
- 招募與培訓計畫  
- 打卡與班表制度  
- 確認以下是否合法
試用期為 15 天，培訓期時間另行通知。
培訓期與試用期皆採時薪計算。
員工於培訓期及試用期內，時薪為 190 元。
若員工完成試用期 15 天，或於培訓期、試用期內經公司終止聘用，
則培訓期與試用期之薪資皆依 200 元／小時 補發及計算。

### 員工行為與服務守則

1. **接待禮儀**  
   - 與客人對眼時，應主動問候：「你好，請問內用還是外帶？」  
   - 保持親切態度，避免臭臉或不耐表情。  

2. **夾鍋與服務細節**  
   - 夾鍋時湯汁若滴出，應立即道歉並擦拭乾淨。  
   - 維持整潔與禮貌是基本服務態度。  

3. **點餐流程**  
   - 若不確定誰先點餐，應禮貌詢問：「請問誰先到呢？」  
   - 讓客人自行決定先後順序，避免爭議。  

4. **上班期間規範**  
   - 外場工作時禁止滑手機與坐下休息。  
   - 私事或身體不適，請先告知再至休息室處理與休息。  
   - 上班時須穿著**包鞋**，以防熱湯潑濺燙傷。  

5. **顧客反應與應對原則**  
   - 若為**廠商食材問題**（如魚頭、臭豆腐等）：  
     回覆「最近沒有遇過這樣的情況，我們會再與廠商確認。」  
   - 若為**本店問題**（如蔬菜變黃、食材異味等）：  
     回覆「會再請後台確認問題原因。」  
   - 不論為何原因，**一律需退錢或更換品項**。  
   - 若確認無問題，仍須禮貌解釋並讓客人理解。  

6. **抽菸與休息規範**  
   - 需在確認**店內有人接應**或工作階段結束後（補鍋、清潔、櫃台顧客有人接應）方可外出。  
   - 禁止於店門口抽菸，應選擇可監看店內狀況之地點。  

7. **煮鍋與工作禮儀**  
   - 煮鍋、夾鍋時不得交談，以免影響衛生或讓客人觀感不佳。  

8. **客訴與特殊狀況**  
   - 若遇到無法判斷或不確定如何回覆的情況，請**直接交由老闆處理**，避免自行應對錯誤。
